# Offers Carousel

The Offers Carousel displays special promotional packages with integrated WhatsApp call-to-action buttons and animated gift badges.

## Location

- **Component**: `components/offersCarousel.tsx`
- **Data**: `components/offers.tsx`

## Features

- Responsive carousel (horizontal cards on desktop, swipeable on mobile)
- WhatsApp direct messaging integration
- Animated spinning text for gift promotions
- Checkmark list for offer details
- Dot indicators for mobile navigation
- Embla Carousel for smooth scrolling

## WhatsApp Integration

### URL Generator

```tsx
const phone = "51903565918";

const getWhatsAppUrl = (message: string) =>
  `https://api.whatsapp.com/send?phone=${phone}&text=${encodeURIComponent(message)}`;
```

### CTA Button

```tsx
<a
  href={getWhatsAppUrl(offer.whatsappMessage)}
  target="_blank"
  rel="noopener noreferrer"
  title={`Chatea sobre ${offer.title} por WhatsApp`}
  aria-label={`Abrir chat de WhatsApp para ${offer.title}`}
  className="w-full"
>
  <button className="w-full py-2 rounded-full bg-[#1F01B9] text-base font-semibold text-white transition-colors hover:bg-[#391FB6]">
    Solicitar ahora
  </button>
</a>
```

### Pre-filled Messages

```tsx
whatsappMessage: "Hola, estoy interesado en el Combo Verano (S/15). ¿Podrías brindarme más información?"
```

Messages are automatically URL-encoded and pre-filled in WhatsApp.

## Performance

- `will-change-transform` optimizes carousel animations
- Pre-encoded WhatsApp URLs for faster navigation
- Optimized images for gift icons
- Touch-friendly tap targets (minimum 44x44px)
