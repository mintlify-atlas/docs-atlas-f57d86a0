# Introduction to Eufrosine

Eufrosine is a modern, high-performance website for a premium water company based in Peru. Built with Next.js 16 and React 19, it showcases the company's products, services, and commitment to quality water distribution.

## What is Eufrosine?

Eufrosine ("Agua de Mesa Eufrosine") is a complete web solution for a water bottling and distribution business. The website features:

- **Premium Product Showcase**: Interactive carousels displaying water bottle presentations (500ml, 625ml, 7L, 20L)
- **Business Information**: Company mission, vision, values, and certifications
- **Contact System**: Integrated contact form with email notifications
- **Smooth Animations**: Lenis smooth scrolling and GSAP animations for enhanced UX
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices
- **Event Gallery**: Showcase of sponsored sports and community events
- **Special Offers**: Dynamic carousel for promotional content

<Note>
  Eufrosine is production-ready and currently deployed at [eufrosine.vercel.app](https://eufrosine.vercel.app/)
</Note>

## Tech Stack

Eufrosine is built with modern web technologies:

### Core Framework
- **Next.js 16.0.10** - React framework with App Router
- **React 19.2.0** - Latest React with concurrent features
- **TypeScript 5** - Type-safe development

### Styling & UI
- **Tailwind CSS 4** - Utility-first CSS framework
- **Radix UI** - Accessible component primitives
- **Lucide React** - Modern icon library
- **Geist Font** - Optimized typography

### Animations & Interactions
- **Framer Motion 12** - Declarative animations
- **GSAP 3** - Professional-grade animation library
- **Lenis** - Smooth scroll experience
- **Embla Carousel** - Touch-friendly carousels

### Backend & Forms
- **Nodemailer** - Email integration for contact forms
- **SweetAlert2** - Beautiful alert modals
- **API Routes** - Serverless functions for form handling

### 3. Contact Form with Email Integration

The contact form uses Next.js API routes to send emails via Nodemailer:

```tsx
const handleSubmit = async (e: React.FormEvent) => {
  e.preventDefault();
  setIsSubmitting(true);

  try {
    const response = await fetch("/api/contact", {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify(formData),
    });

    if (!response.ok) throw new Error("Error sending message");

    await Swal.fire({
      title: "¡Mensaje Enviado!",
      text: "Te responderemos dentro de las 24 horas.",
      icon: "success",
    });

    setFormData({ name: "", email: "", phone: "", message: "" });
  } catch (error) {
    await Swal.fire({
      title: "¡Hubo un error!",
      text: "Por favor, inténtalo nuevamente.",
      icon: "error",
    });
  } finally {
    setIsSubmitting(false);
  }
};
```
