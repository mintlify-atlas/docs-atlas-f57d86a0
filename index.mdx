    Quick Start
  </h2>
  <p className="mb-8 text-base text-gray-600 dark:text-gray-400">
    Get your Eufrosine website up and running in minutes
  </p>

  <Steps>
    <Step title="Clone the repository">
      Clone the Eufrosine project repository to your local machine:
      
      ```bash
      git clone https://github.com/JoseSanchez90/proyecto-eufrosine.git
      cd proyecto-eufrosine
      ```
    </Step>
    
    <Step title="Install dependencies">
      Install all required packages using npm:
      
      ```bash
      npm install
      ```
      
      <Note>
        This project uses Next.js 16 with React 19 and requires Node.js 18 or higher.
      </Note>
    </Step>
    
    <Step title="Configure environment variables">
      Create a `.env.local` file with your email credentials for the contact form:
      
      ```bash
      EMAIL_USER=your-email@gmail.com
      EMAIL_PASSWORD=your-app-password
      ```
    </Step>
    
    <Step title="Start development server">
      Run the development server and open your browser:
      
      ```bash
      npm run dev
      ```
      
      Navigate to `http://localhost:3000` to see your site in action.
    </Step>
  </Steps>
</div>
