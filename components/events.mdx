# Events Section

The Events section displays sponsored events and collaborations with image galleries, descriptions, and hashtag badges.

## Location

Implemented directly in `app/page.tsx` (lines 403-543)

## Features

- Alternating left/right image layouts
- Hover zoom effects on images
- Badge integration for event hashtags
- Responsive grid layout
- Overflow hidden for smooth image transitions
- Neumorphic card shadows

## Hover Effects

### Image Zoom

```css
.group:hover .scale-135 {
  transform: scale(1.35);
}

transition-transform duration-700 ease-out
```

Images scale to 135% with a smooth 700ms easing transition on hover.

### Implementation

```tsx
<div className="overflow-hidden w-full md:w-3/7">
  <div className="transition-transform duration-700 ease-out lg:group-hover:scale-135">
    <img src="/img/evento-1.webp" alt="Event" className="w-full h-full object-cover" />
  </div>
</div>
```

The parent div has `overflow-hidden` to prevent the scaled image from breaking the card layout.

## Responsive Behavior

### Mobile (< 768px)

- Images appear above content
- Single column layout
- Smaller text sizes
- Reduced padding and gaps

### Desktop (>= 768px)

- Side-by-side layout (image + content)
- Image takes 3/7 width, content takes 3/5 width
- Alternating left/right image positions
- Larger text and spacing
- Hover zoom effects enabled

### Layout Classes

```css
/* Mobile */
flex flex-col

/* Desktop - Image Left */
md:flex-row

/* Desktop - Image Right */
md:flex-row-reverse
```

## Potential Refactoring

Consider extracting to a reusable component:

```tsx
// components/eventCard.tsx
interface EventCardProps {
  image: string;
  title: string;
  description: string;
  tags: string[];
  imagePosition?: 'left' | 'right';
}

export function EventCard({ 
  image, 
  title, 
  description, 
  tags, 
  imagePosition = 'left' 
}: EventCardProps) {
  const flexDirection = imagePosition === 'right' ? 'md:flex-row-reverse' : 'md:flex-row';
  
  return (
    <div className={`group flex flex-col ${flexDirection} rounded-[40px] bg-white overflow-hidden shadow-[...]`}>
      <div className="overflow-hidden w-full md:w-3/7">
        <div className="transition-transform duration-700 ease-out lg:group-hover:scale-135">
          <img src={image} alt={title} className="w-full h-full object-cover" />
        </div>
      </div>
      
      <div className="p-4 md:p-6 text-start w-full md:w-3/5 flex flex-col justify-between gap-4">
        <div className="flex flex-col gap-3">
          <h3 className="text-base xl:text-xl font-semibold text-[#1F01B9]">{title}</h3>
          <p className="text-gray-700 text-sm xl:text-base">{description}</p>
        </div>
        <div className="flex items-start gap-2">
          {tags.map((tag) => (
            <Badge key={tag} className="bg-blue-500 text-white text-xs xl:text-sm">
              <FaHashtag className="w-8 h-8" />
              {tag}
            </Badge>
          ))}
        </div>
      </div>
    </div>
  );
}
```
